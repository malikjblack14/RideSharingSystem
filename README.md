# Ride Sharing System (C++ and Smalltalk)

## Overview
This project demonstrates object-oriented programming principles by implementing a Ride Sharing System in both C++ and Smalltalk.

## Concepts Demonstrated
- Encapsulation
- Inheritance
- Polymorphism

## Structure

### C++ Version
- ride.cpp
- Demonstrates class inheritance and virtual functions.

### Smalltalk Version
- RideSharingSystem.st
- Demonstrates message passing, dynamic binding, and subclass behavior.
